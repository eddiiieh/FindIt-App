# 🎒 FindIt – Campus Lost & Found App

**FindIt** is a mobile application designed to help students easily report and recover lost items within a university setting. The app streamlines lost-and-found processes, improves recovery rates, and builds a more connected campus community.

---

## 🚀 Features

- 📸 **Image Upload** – Post pictures of lost/found items  
- 🔍 **Searchable Listings** – Easily browse through reported items  
- 🔔 **Smart Notifications** – Get notified when items match your report  
- ☁️ **Real-time Sync** – Firebase & Supabase-backed cloud database  
- 🔐 **Secure Login** – Firebase Authentication for user access  
- 🎨 **Modern UI** – Clean, mobile-first interface designed with Figma

## 🎯 Purpose

FindIt was created to digitize and simplify the traditional campus lost and found system. By leveraging mobile tech and real-time data sync, the app ensures faster reporting and recovery of items, boosting student convenience and peace of mind.
